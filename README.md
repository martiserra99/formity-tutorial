# Formity Tutorial

Starting point for the [Formity tutorial](https://formity.app/docs/getting-started/tutorial).

## Overview

This repository contains a single-step form built with [React Hook Form](https://react-hook-form.com/). The tutorial walks you through converting it into a dynamic multi-step form with conditional logic using [Formity](https://formity.app).

## Getting started

Clone the repository.

```bash
git clone https://github.com/martiserra99/formity-tutorial
```

Install the dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```

## Next steps

Follow the [tutorial](https://formity.app/docs/getting-started/tutorial) to build the multi-step form step by step.
